    spawn

    Option

    Spawning car

    allows to display a car, which will not be permanent.

    option :

    • - Ride in pilot :
      automatically teleport as a pilot
    • - equipped with nitro :
      Equipped with the default TAB nitro vehicle


    warning :

    the car created does not belong to the RP will therefore not be registered, in addition it has a special license plate to properly identify it
